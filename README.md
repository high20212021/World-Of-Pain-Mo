## 下载  
前往Github Releases下载即可。  
如果你需要开发版，请前往Github Actions(需要Github账户)。  
**目前处于早期开发阶段。**  
## 构建  
### Linux  
提前准备：Git,OpenJDK 17  
克隆仓库后，运行./gradlew build进行构建  
在build/libs可以看到构建之后的文件  
### Windows  
提前准备：Git,Java 17  
克隆仓库后，在仓库位置打开命令提示符，运行gradlew.bat build进行构建  
在build/libs可以看到构建之后的文件
